# Job Fit Evaluation Rubric (mandatory)

Score each dimension 0-100 against the candidate profile:

## 1. Skills match (weight 30%)
- 80-100: core requirements are the candidate's primary skills
- 60-79: most requirements match, 1-2 learnable gaps
- 40-59: partial match, significant upskilling needed
- 0-39: fundamental mismatch

## 2. Experience match (weight 25%)
- 80-100: direct experience, same domain and role type
- 60-79: related experience, transferable skills clear
- 40-59: adjacent experience, case must be made
- 0-39: unrelated

## 3. Culture and conditions fit (weight 15%)
Role type (remote/hybrid/onsite), seniority, work style vs the candidate's stated preferences. Flag red flags visible in the posting: vague pay, "unpaid internship", unrealistic scope, personal-email-only contact.

## 4. Career alignment (weight 30%)
Does this advance the candidate's stated goals? Would the daily tasks energize or drain them?
- 80-100: strongly aligned, clear growth path
- 40-59: decent job, doesn't build toward goals
- 0-39: dead end or backwards step

## Location/logistics gate (pass/fail, not weighted)
Fail if the role requires relocation or hours incompatible with the candidate's stated location/preference.

## Verdicts (from the weighted average)
- 75+: Strong Fit — definitely apply, tailor everything
- 60-74: Good Fit — apply, address gaps in the cover letter
- 45-59: Moderate Fit — apply only with a strategic reason
- 30-44: Weak Fit — probably skip
- <30: Poor Fit — skip

## Honesty rules
- Score against evidence in the profile, not optimism. A missing skill is a gap even if "learnable".
- If experience match < 50, say explicitly that extensive reframing would be needed and flag which claims would be a stretch.
- Name the 2-3 biggest gaps in every evaluation, not just strengths.
